How to Run the Project
Step 1: Open in Google Colab

Upload the .ipynb file or use Colab link.

Step 2: Run Cell 1 (Disease Detection)
Upload an image when prompted
View predictions
Step 3: Run Cell 2 (Soil Prediction)
Uses sample dataset
Outputs predicted soil moisture
Step 4: Run Cell 3 (Automation)
Simulates irrigation system based on moisture
